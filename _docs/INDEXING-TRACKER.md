## How to Update This File

### When Publishing a New Article:
1. Move the article from "Scheduled" to "Pending Indexing"
2. Add full URL: `https://senticmoney.com/blog/[slug]`
3. Set Published to "Live"
4. Set Requested to `[ ]`

### After Requesting Indexing:
1. Change `[ ]` to `[x]` in the Requested column
2. Add date in Notes if desired

### After Confirmed Indexed:
1. Move URL from "Pending" to "Indexed Pages" table
2. Add the indexed date from Google Search Console
